# Shop Inventory Management System (C++ & SQLite)

A console-based inventory management system built using **C++ STL** and **SQLite**.

## Features
- Add vegetables and fruits
- Prevent duplicate items using STL map
- Sell items (reduce quantity)
- Remove items completely
- Persistent storage using SQLite
- Sorted display by quantity

## Technologies Used
- C++ (STL: map, vector)
- SQLite (C library)
- g++ compiler

## How to Build
bash :-
g++ shop_inventory.cpp -x c sqlite3.c -o shop_inventory

## How to run
./shop_inventory
